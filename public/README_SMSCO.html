منصة SMSCO.AI

مرحبًا 👋
هذا المستودع يحتوي على منصة الشركة السعودية للخدمات الإدارية مع دعم متعدد اللغات وتحسينات إدارية متقدمة.

⚙️ خطوات الإعداد والتشغيل

1. المتطلبات

  • Node.js 18 أو أحدث
  • NPM
  • قاعدة بيانات SQLite (تلقائية مع Prisma)

2. تحديث الحزم وتشغيل المشروع

نفّذ السكربت التالي من داخل مجلد المشروع:

bash bash setup_and_run_smsco.sh

سيقوم هذا السكربت تلقائيًا بـ:

  • ✅ تحديث package.json
  • ✅ تثبيت جميع الحزم (بما فيها recharts)
  • ✅ تشغيل السيرفر المحلي على http://localhost:3000

3. معلومات الدخول إلى صفحة الإيميلات الإدارية

  • الرابط: /admin/emails
  • كلمة المرور: smsco123

📦 السكربتات المرفقة

| الملف | الوظيفة | |-------|---------| | setup_and_run_smsco.sh | تحديث المشروع وتشغيله دفعة واحدة | | start_smsco_project.sh | تشغيل المشروع فقط | | install_updated_packages.sh | تثبيت الحزم فقط | | backup_emails.sh | حفظ نسخة احتياطية | | restore_emails.sh | استعادة النسخة الاحتياطية |

🧠 الدعم الفني

لأي مساعدة أو تطوير إضافي، يرجى التواصل مع فريق التقنية أو مسؤول المشروع.
